# 🔐 Password Generator (Python)

A simple Python-based password generator that creates strong and random passwords using **letters, numbers, and special characters**.

---

## 📌 Features

- Generates random passwords of user-defined length
- Includes:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (!@#)
- Beginner-friendly logic
- Uses Python built-in modules (`random`, `string`)
